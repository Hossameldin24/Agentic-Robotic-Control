# Agentic Robotic Control

This repository contains implementations for agentic robotic control systems, combining LLM-based planning with robotic manipulation.

## Projects

### Inner_Monologue
An implementation of intelligent robotic control using inner monologue techniques with tool orchestration for robotic tasks.

### LLM-TAMP
Large Language Model Task and Motion Planning (LLM-TAMP) system for robotic manipulation tasks.

## Getting Started

Each project has its own setup instructions and requirements. Please refer to the individual project directories for specific setup and usage instructions.

## Features

- Context-aware robotic planning
- Human-in-the-loop scene understanding
- Intelligent tool orchestration
- Multi-modal environment interaction
- Dynamic task planning and execution

## Requirements

- Python 3.8+
- PyBullet for physics simulation
- Various LLM APIs for planning and reasoning
